🎮 Bevy 2D Platformer (WASM)

Controls

Move Left: A or ←
Move Right: D or →
Jump: Space
Fullscreen: F11

How to Play

Blue square is you! Other players appear as red squares. Push the brown boxes around and jump on them to reach higher places.

Server must be running at ws://127.0.0.1:8080. Start it with:
cargo run --example bevy_platformer_server
